## Description

Joking aside, this project's intent is to demonstrate react routing, react redux, and a dash of Material UI by creating a survey workflow. In addition there is an admin page for managing the submissions.

## Technologies Used

 - React
 - React-Redux
 - Express.js
 - Material UI
 - Axios
 - Postgres Driver
